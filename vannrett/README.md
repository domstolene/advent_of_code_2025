# VannRett advent of code 2025
## Kom i gang

1. Åpne mappen _vannrett_ som nytt prosjekt i IntelliJ. 
2. Høyreklikk på _src/main_ -> **Mark directory as** -> **Sources Root**.
3. Høyreklikk på _src/resources_ -> **Mark directory as** -> **Resources Root**.
4. Legg din input for første dag i src/resources/input_dag_1.txt.
5. Prøv fra start-menyen, prøv å kjøre Dag_1Kt.
